# travel-logger
A full stack application for listing and storing the places you have traveled.

## DB modeling
- Log entry
*Title - Text
*Description - Text
*Comments - Text
*Rating - scal of 1 - 10
*Image - Text - URL
*Start Date - DateTime
*End Date - DateTime
*Latitude - Number 
*Longitude - Number
*Created at - DateTime
*Updated at - DateTime
